Page Layout with Semantic HTML Markup

This example demonstrates how to build a web page using semantic elements. HTML5 provides tags like <header>, <nav>, <main>, <section>, <article>, <aside>, and <footer> to give your pages meaningful structure.

Static web pages are structured with statements:
  1. <!DOCTYPE html> declaration
  2. <html lang="en"> ... </html> element that contains the page's <head> and <body> sections.
  3. <head> ... </head> containing:
    1. A sequence of meta statements
    2. <title> [a title] </title>
    3. a sequence of in-line and linked scripts and styles
  4. <body> ... </body> containing all the page's html markup and may optionally contain, at the end, a <script> section that creates JavaScript objects and may attach event handling to specific html elements.
HTML elements with symantic meaning make pages easier to read, understand, and maintain.

Table 1 - Semantic Markup Elements

Element Description and Example
<header> Introductory content or navigation links.
<header>Site Title</header>
<nav> Navigation links section.
<nav><a href="#">Home</a></nav>
<main> Main content unique to the document.
<main><article>...</article></main>
<section> Standalone section of related content.
<section><h2>Features</h2></section>
<article> Independent, self-contained content.
<article><h2>News</h2></article>
<aside> Secondary content, like a sidebar.
<aside>Related links</aside>
<footer> Footer for section or page.
<footer>© 2025</footer>
<h1>–<h6> Headings indicating content hierarchy.
<h2>Section Title</h2>
<address> Contact information.
<address>Email us</address>
<figure> Image or illustration with caption.
<figure><img src="..."><figcaption>Caption</figcaption></figure>
<figcaption> Caption for a figure.
<figcaption>Figure 1: Chart</figcaption>
<time> Machine-readable time/date.
<time datetime="2025-08-03">Aug 3, 2025</time>
<mark> Highlight or mark text.
Use the <mark>important</mark> tag.
<summary> Visible heading for <details>.
<details><summary>More</summary>...</details>
<details> Disclosure widget for showing/hiding content.
<details><summary>Info</summary>Hidden content</details>
<blockquote> Quoted section from another source.
<blockquote cite="url">Quote text</blockquote>
<cite> Title of work (e.g., book, article).
<cite>Moby Dick</cite>
<code> Computer code snippet.
<code>let x = 10;</code>
