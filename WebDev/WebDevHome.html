WebDev focuses on technologies for building static websites, using only HTML, CSS, and JavaScript. One of the prime motives for building this website is to explore ways of presenting text, diagrams, and code in ways that support learning and provide reference ideas, tables, and examples. We examine ways of managing real estate by showing or hiding code, providing diagrams and images that users can resize and widgets like vertical splitters that hold code on each side that often overflow the display space until the user expands it to the left or right. The site also experiments with content organization into Tracks, Stories, Bites, and examples. A row of buttons near the top of each page provides access to language and technology tracks. Each track has an explorer with a left panel of links to track resources and right panel showing content accessed from the left panel. You are looking at the WebDev Explorer with its Track Summary in the right panel. Each page displays links at the bottom that provide information about the page and place the page in a thread of related material with "Next" and "Prev" buttons. Most of the WebDev content is delivered in the form of bites. They contain relatively brief discussions of the static web technologies:
  • Hyper Text Markup Language (HTML)
  • HTML Document Object model (DOM)
  • Cascading Style Sheets (CSS)
  • JavaScript (JS)
  • Examples of page constructions used in this site
  • Examples of Widgets also used in the site
 Eventually, discussions about using Asp.Net to build pages dynamically will be added, but there are none of those discussions at this time.
