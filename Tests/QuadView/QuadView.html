Setup Instructions
This is a prototype web viewer. To view content start a local server:
python -m http.server 8000
at the root directory holding files to view. Then, load an HTML page using browser address:
http://localhost:8000/MyPage.html
Then load html, css, and js urls into the input controls using paths relative to the loaded page
