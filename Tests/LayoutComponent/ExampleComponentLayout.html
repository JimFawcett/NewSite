Introduction

This area demonstrates the <x-two-panel> component. The container has a fixed visual height here so you can clearly see both panels.

Usage

<x-two-panel left="22rem" gap="1rem" stack-bp="900px">
  <div slot="left">...</div>
  <div slot="right">...</div>
</x-two-panel>

Theming

  • --two-left (panel width)
  • --two-gap (gap size)
  • --two-border, --two-radius
