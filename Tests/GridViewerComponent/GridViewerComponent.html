GridViewer Component

This is a W3C component that implements a two-panel view of HTML content. It has a grab-bar in the middle to change the relative sizes of the panels. Panel size is persistent over page reloads, but can be reset to the default values with an onclick="resetTwoPanel('panel-sizes'). The component is used like any HTML element and can be styled in the usual way, e.g., as shown in the content of this page. Use the browser's "view page source" to see that.
