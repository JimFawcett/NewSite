1.0 Introduction

Code Track provides a place to build and display example code snippets, projects, and tools for code generation. Experiments and demos written in Rust, C++, C#, and Python reside here. It is a place to:
  • experiment with AI support for code development and maintenance
  • Compare implementations of projects in more than one language, e.g., size, complexity, maintainability
  • Build smallish tools for code management
  • Try new language features

2.0 Contents

CodeBites - short focused pages on using AI for code development:
  • Chat bots (ChatGPT, Claude, Gemini) - prompting for analysis, generation, documentation
  • Code AI CLI sessions - using Claude Code and Gemini CLI interactively
  • AI agents - building and running agents with the Anthropic API
  • Agentic AI - multi-step autonomous workflows
  • LLM API - calling language models directly from code
  • Skills AI - adding tools and skills to agents
  • Spec-driven development - using Constitution.md, Structure.md, and Spec.md files with AI
 Projects - multi-language implementations of two projects:
  • TextFinder - searches directory trees for files containing a specified text pattern; implemented in Rust, RustOpt, C++23, C#, and Python
  • PageValidator - validates HTML files for structural correctness; implemented in Rust, C++, C#, and Python
  • Utilities: tf_timer.py, pa_timer.py (performance timing), code_metrics.py (line counts)
 AI Demos - runnable agents and CLI sessions under the AI/ folder:
  • DemoAgent-Claude, DemoAgent-Claude2 - readme generation agents using Anthropic API
  • DemoCode-Claude, DemoCode-Gemini - code generation demos
  • CppCodeAnalysis, RustCodeAnalysis - language-specific analysis agents
  • Scripts, Skills - reusable agent utilities
 DesignStructure - Rust implementations of software design patterns:
  • BasicStructure, FactoredStructure, DataFlowStructure
  • PlugInDataFlowStructure, PlugInWithTraitObjects
  • TypeEraseDataFlowStructure, TypeEraseDataFlowStructureWithTraitObjects
 CodeWebifier - .NET tool that converts source code files to syntax-highlighted HTML pages.
