Code/AI - Directory Contents

One-line survey of each subdirectory under Code/AI/

Folder Description
Claude Context management notes and configuration docs for Claude Code - covers context levels, settings, slash commands, and prompt-phrasing guidance for controlling output behavior.
CppCodeAnalysis Static analysis output for a C++ ThreadPool project - reports per-file and per-function line counts and complexity metrics in tabular HTML.
DemoAgent-Claude First-generation interactive code assistant - indexes a source directory and answers conversational queries about the code via the Anthropic API.
DemoAgent-Claude2 Second-generation interactive dev agent with commands /analyze, /improve, /readme, /tree, /files, /clear, /quit. Generates improvement suggestions and README drafts from a source directory.
DemoAIConsole-Gemini Session transcripts from an interactive Gemini console loop showing example AI-assisted command and code execution exchanges.
DemoCode-Claude Claude Code workspace configuration only; placeholder for future Claude API code examples.
DemoCode-Gemini Hello World programs in C++ and Rust generated using the Gemini CLI, with session transcripts showing the generation workflow.
DemoLibraryAgent-Claude Agent that reads library source files, extracts the public interface (functions, classes, types), and generates a complete runnable demonstration application in the library's own language.
ReadmeAgent-Claude Archive of the original README generator agent; superseded by the version in Scripts/.
RustCodeAnalysis Static analysis output for Rust crates - reports per-file and per-function line counts and complexity metrics in tabular HTML.
RustDemoAgent-Claude Agent tuned for Rust crates: generates markdown API documentation and working Rust example programs from a crate's source files.
Scripts Utility agents: a dev agent for code analysis, a README generator using customizable templates, and an implementation-smell detector.
Skills HTML reference catalog of implementation smells organized by category, with symptom descriptions and impact ratings; imported by agent tools as reusable context.
