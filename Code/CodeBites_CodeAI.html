1.0 - Introduction

This page is just starting. It will eventually contain examples of AI coding platform useage. Claude code is a command-line tool that gives the claude.ai access to local development environment. It can read local code, make edits, and run commands. This supports a flexible means of development for local repositories without writing any agent code.
  1. Analyze project structure and report size, complexity, and code smells.
  2. Refactor code on new branch, test and review, and commit. Merge with main branch as appropriate. Claude can help with merge clashes.
  3. Build unit test code, run, and resolve issues.
  4. Document public interfaces and project specification. The first is generated by the AI. The second should be something close to a list of all the creational prompts.
Agents can do the same thing, but you have to write agent source (usually small and simple) for each part of the development process.
