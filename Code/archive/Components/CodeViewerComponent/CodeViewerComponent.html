*/
#![allow(dead_code)]
#![allow(clippy::approx_constant)]

use std::fmt::*;

fn main() {
println!("\nRust Basics");
primitives();
libtypes();
usertypes();
}
This is a code viewer component, wrapped in a div that floats left and pads the figure from its surrounding text. Clicking on the code makes the figure larger and clicking on the title makes it smaller. The title text will wrap if the figure body gets small enought that it does not fit into the title box unwrapped. All the text in this demonstration is placed within <t-b> blocks. That limits line lengths to 50 rem, for readability.
